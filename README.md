# 智扫通机器人智能客服

## 项目介绍

这是一个基于Streamlit和LangChain的智能客服项目，专注于扫地机器人领域的专业知识问答。

## 功能特性

- **智能对话**：基于大语言模型的自然语言交互
- **知识检索**：使用RAG技术从知识库中检索相关信息
- **工具集成**：支持天气查询、用户信息获取等工具
- **个性化报告**：生成用户使用情况报告
- **流式输出**：实时显示回答内容

## 技术栈

- Python
- Streamlit
- LangChain
- Chroma (向量数据库)
- 通义千问 (大语言模型)

## 项目结构

```
PythonProject/
├── agent/             # 智能代理模块
├── chroma_db/         # 向量数据库
├── config/            # 配置文件
├── data/              # 知识库数据
├── logs/              # 日志文件
├── model/             # 模型工厂
├── prompts/           # 提示词模板
├── rag/               # RAG服务
├── utils/             # 工具函数
├── app.py             # 主应用
└── README.md          # 项目说明
```

## 运行方法

1. 安装依赖
2. 运行 `streamlit run app.py`
3. 在浏览器中访问 http://localhost:8501
